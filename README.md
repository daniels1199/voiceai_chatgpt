# voiceai_chatgpt
This project try to emulate a API with voice assistant using ChatGPT engine.

# Requirements
  - a linux environment
  - python3

## Packages to install:

By apt:
 - speech-dispatcher
 - speech-dispacher-rhvoice
 - python3-speechd
 - rhvoice
 - rhvoice-brazilian-portuguese #"Leticia-F123" (or any language you want)

By pip:
 - dotenv
 - openai
 - SpeechRecognition
